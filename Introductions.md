# ROBLOX_PROJS

Test1 is a finished game, its gameplay mainly revolves around collecting objects and selling them for upgrades.
Players go around the map to collect falling objects to sell them for currency, which is used for various purposes (upgrading, accessing new biomes)

------------------------------------------------------------------------------------------------------
Test2 is an unfinished game, featuring random generated dungeons system (though still unfinished) and a weapon system ( player can equip or access different weapons in their arsenal by pressing R and left_mouse_button to use them).

Both games use module scripts to store and access data for easier expansion and managing resources.
